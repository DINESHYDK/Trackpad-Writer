The insight: why hold a click manually when software can do it? I wrote an AutoHotKey script that maps CapsLock to toggle the left mouse button. Press once → LButton held DOWN (drawing starts). Press again → LButton UP (drawing stops). Mouse-free, keyboard-free writing.

Setup Guide

AutoHotKey Setup —
Done in 5 Minutes.

Follow these steps once. After that, the script runs automatically in the background every time you need it.

1
Download AutoHotKey v1.1
Visit autohotkey.com/download/1.1/ and download the latest v1.1 installer. Run it with default settings. Use v1.1 specifically — the script below is written for v1.1 syntax, not v2.
2
Create a New Script File
Right-click anywhere on your Desktop → New → AutoHotKey Script. Name it trackpad-writer.ahk. This creates a blank .ahk file ready to edit.
3
Paste the Script
Right-click the .ahk file → Edit Script. Delete all existing placeholder text. Paste the full script from the code block below. Save the file with Ctrl + S.
4
Run the Script
Double-click the trackpad-writer.ahk file. A green H icon appears in your system tray (bottom-right corner of taskbar). The script is now running. CapsLock will now toggle drawing mode.
5
Optional: Auto-Run on Windows Startup
Press Win + R → type shell:startup → press Enter. A folder opens. Paste a shortcut to your .ahk file here. The script now loads automatically every time Windows starts.
trackpad-writer.ahk
; ============================================
; TrackPad Writer — AutoHotKey Script
; Turns CapsLock into a drawing-mode toggle
; Compatible with AutoHotKey v1.1
; ============================================
;
; HOW IT WORKS:
;   Press CapsLock ONCE  → Holds left click   (Start drawing)
;   Press CapsLock AGAIN → Releases left click (Stop drawing)
;   Press Escape         → Emergency release at any time
;
; USAGE:
;   1. Open any drawing app (Whiteboard, Paint, OneNote, Excalidraw)
;   2. Select the pen/draw tool in the app
;   3. Press CapsLock → Drawing Mode ON (tooltip appears)
;   4. Write with your capacitive stylus on the trackpad
;   5. Press CapsLock again → Drawing Mode OFF
; ============================================

#NoEnv
#SingleInstance Force
SendMode Input
SetWorkingDir %A_ScriptDir%

; State variable — tracks whether click is currently held
clickHeld := false

CapsLock::
    if (clickHeld) {
        ; Release the held mouse button — stop drawing
        Send {LButton up}
        clickHeld := false
        ToolTip, ✋  Drawing Mode  OFF,, , 1
    } else {
        ; Hold the mouse button down — start drawing
        Send {LButton down}
        clickHeld := true
        ToolTip, ✏️  Drawing Mode  ON,, , 1
    }
    ; Auto-hide the tooltip after 1.5 seconds
    SetTimer, RemoveTooltip, -1500
return

RemoveTooltip:
    ToolTip,,,,1
return

; ---- Emergency release ----
; Press Escape at any time to instantly stop drawing
Escape::
    if (clickHeld) {
        Send {LButton up}
        clickHeld := false
        ToolTip, ✋  Released,, , 1
        SetTimer, RemoveTooltip, -1000
    }
return
⚠️  Important: This script uses AutoHotKey v1.1 syntax, which differs from v2. If you already have AHK v2 installed, download v1.1 from autohotkey.com/download/1.1/ and use it to run this file.
How to Use

The 4-Step Flow.

Once the script is running, open any drawing app and follow this process every time you want to write.

01
Open a Drawing App
Launch Whiteboard, Paint, OneNote, or any app. Select its Pen / Draw tool from the toolbar.
02
Press CapsLock
A tooltip shows "✏️ Drawing Mode ON". Left click is now held. The canvas is ready.
03
Write with Stylus
Move the capacitive stylus on your trackpad. Ink strokes appear in real time. Write naturally.
04
Press CapsLock Again
Tooltip shows "✋ Drawing Mode OFF". Click released. Lift the pen between letters to break strokes.
💡 Pro Tips
→ Press Escape anytime to instantly release the click if anything goes wrong.
→ Use Ctrl + Z in your drawing app to undo any accidental strokes.
→ Rest your hand on the laptop chassis, not the trackpad, to avoid palm-rejection issues.
→ In Microsoft Whiteboard, select the Pen tool before pressing CapsLock.
